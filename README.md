# E-Commerce Apparel Sales Analysis (2022)

##  Project Overview
This project analyzes 2022 sales data of an online apparel retailer to understand customer behavior, sales trends, and revenue drivers using PostgreSQL and Excel.

##  Objectives
- Identify top-performing months, channels, and product categories
- Analyze customer demographics and purchasing patterns
- Evaluate geographic and channel-based revenue contribution
- Support data-driven business decisions through visual dashboards

##  Dataset
- ~31,000 transaction records
- Time period: Janâ€“Dec 2022
- Key fields: Orders, Customers, Products, Channels, Geography, Revenue

## Tools & Technologies
- PostgreSQL (SQL, CTEs, Window Functions)
- Microsoft Excel (Pivot Tables, Dashboards)
- GitHub (Version Control)

##  Key Analysis Performed
- Monthly sales and order trends
- Gender and age group contribution analysis
- Channel and category performance
- Top states and cities by revenue
- Repeat vs new customer analysis
- Average Order Value (AOV) by channel

##  Dashboard
An interactive Excel dashboard was created to visualize insights and enable business-friendly analysis.


##  Key Insights
- March recorded the highest sales and order volume
- Women customers contributed ~64% of total revenue
- Amazon, Myntra, and Flipkart generated ~80% of sales
- Maharashtra, Karnataka, and Uttar Pradesh were top revenue states

##  Note:
The dataset used is from a public open-source repository and has been anonymized for learning and analysis purposes.
